# 新月Wiki

新月のWiki的なドキュメントです。

## 新月とは
新月とはいくつかのサーバーで動いているスレッドフロート型掲示板の集合体です。
その中のいくつかのスレッド(掲示板)はP2Pの仕組みによってデータを共有しています。
ソフトウェアをダウンロードして掲示板を運営することもできます。
